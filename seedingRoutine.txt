===================================================================
Seeding behaviour
===================================================================
• The seeder creates the target user if they don't exist (by email).
• Re-running is idempotent: ALL of the user's data is wiped and
  re-seeded by default. Pass `--keep-personal` to preserve journal
  entries, notes, water entries, and tags across re-seeds.
• All habits start on the sprint start date (29 Apr 2026).
• Habit-tasks are NOT pre-created. They auto-generate on the day
  itself via `POST /sync/habits/{user_id}`, which the Dashboard and
  Tasks pages call on mount. This keeps the Tasks view uncluttered.
• Water goal is set to 3000 ml.
• Reminder config defaults to: 1 day before, on due date, when overdue.
